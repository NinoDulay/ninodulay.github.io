001 Featured Project

Shopify Discount Deal Monitor

E-Commerce Client

I built a Python bot that monitors multiple Shopify store links by fetching their product JSON endpoints at regular intervals. It uses jsondiff to compare the latest product data against what was previously saved in SQLite — the moment a price drop or discount is detected, it fires an alert straight to a Slack channel via webhook. Ran continuously for nearly 2 years and saved the company a significant amount by catching deals in seconds that would have otherwise been missed.

Business Impact
Saved the company an estimated ₱500K+ over ~2 years in procurement costs by capturing discounted deals within seconds of appearing. Monitors 100+ products simultaneously with near-zero false positives.

002HR Automation

Pre-Employment Email Automation

Confidential Client

I built a Python automation that reads a list of new hires from an Excel file using openpyxl, then uses Jinja2 to render a personalized HTML email for each person with the correct onboarding documents attached. The tricky part was that there are 5 site locations — Location 1 through Location 5 — each with their own email template and set of PDF attachments, so the script automatically routes each hire to the right variant based on their assigned site. Emails are sent through the SendGrid API, and the whole thing runs automatically every day via the schedule library — no manual work needed. It also backs up the source file after each run.

Business Impact
Eliminated ~6 hrs/day of manual email prep across 5 sites. Near-zero human error on attachment selection.
PythonSendGrid APIJinja2PandasScheduleopenpyxlHTML Emaildotenv

008Process Automation

Change Request Lead Time Enforcer

Confidential Client

IT change management requires at least 6 hours of lead time before any change can proceed — but people were manually checking this for every request, which took hours given the volume of RFCs coming in. I built a Power Automate flow that automatically queries ServiceNow via its API to fetch filtered change requests, calculates whether each one meets the 6-hour lead time requirement, and if it doesn't — it cancels the associated Outlook Calendar meeting via Microsoft Graph API and sends a Slack notification to the team. It also writes a withdrawal note directly back into the ServiceNow RM notes so everything is logged in one place.

Business Impact
Eliminated hours of manual lead time checks per day. Non-compliant changes are now caught and cancelled automatically — no human intervention needed.
